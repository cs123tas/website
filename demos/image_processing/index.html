Image Processing Demo

In this demo, you can experiment with different reconstruction filters for image scaling.

In order to use the demo, follow these steps:
  1. Use the slider to set the number of pixels in the output image width and height. Mess around with this to see the effect it has on the outputs before moving on
  2. Sketch the two different functions you want to use for scaling (or use pre-defined ones)
  3. Click and drag on the mandrill to select a region you want to scale
  4. Click apply filter on the top and bottom graphs in order to see the results
